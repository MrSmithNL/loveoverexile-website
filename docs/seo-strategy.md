## Image SEO Standards

### File Naming Convention

**Format:** `[descriptive-phrase]-[context].jpg`
- Words separated by hyphens — never underscores, never spaces
- 3–6 words maximum
- Include primary keyword naturally where it fits
- No generic names (`image1.jpg`, `photo.jpg`, `DSC_0049.jpg`)

**Examples:**
```
parental-alienation-recovery-path.jpg         ✅
alienated-parent-journaling-therapy.jpg       ✅
malcolm-smith-author-love-over-exile.jpg      ✅
healing-parent-child-relationship.jpg         ✅
parental-alienation-support-community.jpg     ✅
DSC_0049.jpg                                  ❌
photo1.jpg                                    ❌
image-hero.jpg                                ❌
```

**Site images — agreed filenames:**

| File | What it shows | Filename |
|------|--------------|----------|
| Hero background | Pathway / nature / hope | `hero-healing-path.jpg` |
| Author portrait | Malcolm Smith | `malcolm-smith-author-love-over-exile.jpg` |
| Book cover | Book cover design | `love-over-exile-book-cover.jpg` |
| Community image | Warmth / connection | `alienated-parent-support-community.jpg` |
| Guide cover | Survival guide document | `parental-alienation-survival-guide-cover.jpg` |
| OG default | Brand / abstract | `love-over-exile-og-default.jpg` |

### Alt Text Standards

**Rules:**
- Describe what is *in* the image — don't start with "Image of" or "Photo of" (screen readers say that automatically)
- 80–125 characters ideal
- Include a relevant keyword *only if it fits naturally* — never force it
- End with a period
- If image is purely decorative (no information value), use `alt=""` — screen readers skip it
- Context matters — the same photo gets different alt text on different pages

**Examples — good:**
```
alt="Father sitting quietly at a window, holding a photograph of his child."
alt="Open journal with handwritten notes about healing from parental alienation."
alt="Malcolm Smith, author of Love Over Exile, speaking at a parental alienation awareness event."
alt="Winding forest path leading toward light, representing the healing journey."
```

**Examples — bad:**
```
alt="parental alienation parent child"          ❌ (keyword stuffing)
alt="Image of man"                              ❌ (too vague, starts with "Image of")
alt="photo"                                     ❌ (useless)
alt=""  (on a meaningful image)                 ❌ (skipped by screen readers when it shouldn't be)
```

**Decorative images** (background textures, dividers, abstract shapes) → `alt=""`
